# Coffee_SalesAnalysing
This project presents an interactive Coffee Sales Dashboard developed using Microsoft Power BI to analyze and visualize coffee sales performance. The dashboard provides meaningful insights into sales trends, customer behavior, and product performance, enabling data-driven decision making.

# Objective

The main objective of this project is to:

Analyze coffee sales data across different coffee types, roasts, sizes, and customers

Identify sales patterns and trends over time

Create a professional and reusable dashboard for business analysis

# Dataset Description

The dataset includes the following fields:

Date – Transaction date

Customer – Customer name or ID

Coffee Type – Type of coffee sold

Roast – Roast category (Light, Medium, Dark)

Size – Cup size

Quantity – Number of units sold

Total Sales – Total sales amount

The data was cleaned and transformed using Power Query before visualization.

# Tools & Technologies

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (Calculated measures & KPIs)

CSV Dataset

# Key Features of the Dashboard

Interactive slicers for Date, Coffee Type, Roast, and Size

KPIs for Total Sales, Total Quantity, and Average Sales

Sales trend analysis over time

Comparison of coffee types and roast performance

Clean, professional layout with consistent color theme

# Insights Generated

Identification of top-selling coffee types and roasts

Sales performance trends across different periods

Customer purchase behavior based on coffee size and type

# Conclusion

This project demonstrates practical skills in data analysis, visualization, and dashboard design using Power BI. It showcases the ability to transform raw data into meaningful business insights through interactive visuals.
